-- Descripción del programa --

=> Registra datos introducidos por el usuario sobre planetas y satélites en ficheros de datos.
=> Lee los datos de los ficheros.
=> Genera archivos html con la información obtenida por el usuario al crear los planetas/satélites.
=> Ejecuta los archivos en la web desde el navegador predeterminado.
=> Elimina los archivos.

El directorio raíz contiene los directorios:
- Diagramas: Diseño del programa (Casos de uso y clases)
- Planets: Carpeta del programa en NetBeans.

Es recomendable ejecutar la opción de eliminar archivos al empezar para testear el programa desde 0.

Versión: 1.0 (Sin testeo ajeno).
